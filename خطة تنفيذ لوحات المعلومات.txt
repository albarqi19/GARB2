### 2. خطة التنفيذ التفصيلية:

#### أ. إنشاء لوحة معلومات المساجد والحلقات:

1. إنشاء صفحة لوحة معلومات جديدة:
   ```bash
   php artisan make:filament-page MosquesAndCirclesDashboard --panel=admin
   ```

2. تطوير الويدجات التالية:"تم الفيذ"
   - ويدجت توزيع المساجد حسب المناطق (رسم بياني دائري)
   - ويدجت إحصاءات الحلقات حسب النوع والفترة (رسم بياني شريطي)
   - ويدجت أعداد الحلقات في كل مسجد (جدول)
   - ويدجت معدلات الإشغال (رسم بياني خطي)

#### ب. إنشاء لوحة معلومات المعلمين والطلاب:"تم التنفيذ"

1. إنشاء صفحة لوحة معلومات جديدة:
   ```bash
   php artisan make:filament-page TeachersAndStudentsDashboard --panel=admin
   ```

2. تطوير الويدجات التالية:
   - ويدجت توزيع المعلمين حسب المساجد (رسم بياني دائري)
   - ويدجت إحصاءات الطلاب حسب العمر والمستوى (رسم بياني شريطي)
   - ويدجت نسب الحضور والغياب (مقياس دائري)
   - ويدجت أعلى المعلمين أداءً (جدول)
   - ويدجت أعلى الطلاب حفظًا (جدول)

#### ج. إنشاء لوحة معلومات الرواتب والمالية:

1. إنشاء صفحة لوحة معلومات جديدة:
   ```bash
   php artisan make:filament-page SalariesDashboard --panel=admin
   ```

2. تطوير الويدجات التالية:
   - ويدجت إجمالي الرواتب الشهرية (مؤشر رقمي)
   - ويدجت توزيع الرواتب حسب المساجد (رسم بياني دائري)
   - ويدجت تطور الرواتب خلال العام (رسم بياني خطي)
   - ويدجت الرواتب الإضافية والحوافز (جدول)

#### د. إنشاء لوحة معلومات الإيرادات والمصروفات:

1. إنشاء صفحة لوحة معلومات جديدة:
   ```bash
   php artisan make:filament-page RevenueAndExpensesDashboard --panel=admin
   ```

2. تطوير الويدجات التالية:
   - ويدجت مقارنة الإيرادات والمصروفات الشهرية (رسم بياني مزدوج)
   - ويدجت توزيع المصروفات حسب النوع (رسم بياني دائري)
   - ويدجت توزيع الإيرادات حسب المصدر (رسم بياني دائري)
   - ويدجت التدفق النقدي (رسم بياني خطي تراكمي)

#### هـ. إنشاء لوحة معلومات الطلبات والخدمات:

1. إنشاء صفحة لوحة معلومات جديدة:
   ```bash
   php artisan make:filament-page RequestsAndServicesDashboard --panel=admin
   ```

2. تطوير الويدجات التالية:
   - ويدجت إحصاءات الطلبات حسب النوع (رسم بياني شريطي)
   - ويدجت الطلبات حسب الحالة (مقياس دائري)
   - ويدجت متوسط زمن معالجة الطلبات (مؤشر رقمي)
   - ويدجت آخر الطلبات المقدمة (جدول)

#### و. إنشاء لوحة معلومات التسويق ومؤشرات الأداء:

1. إنشاء صفحة لوحة معلومات جديدة:
   ```bash
   php artisan make:filament-page MarketingAndKpiDashboard --panel=admin
   ```

2. تطوير الويدجات التالية:
   - ويدجت مؤشرات الأداء الرئيسية (بطاقات KPI)
   - ويدجت الأنشطة التسويقية الشهرية (رسم بياني شريطي)
   - ويدجت تحليل متابعي الفروع (رسم بياني خطي)
   - ويدجت فعالية الحملات التسويقية (جدول)

#### ز. إنشاء لوحة معلومات المهام والخطة الاستراتيجية:

1. إنشاء صفحة لوحة معلومات جديدة:
   ```bash
   php artisan make:filament-page TasksAndStrategyDashboard --panel=admin
   ```

2. تطوير الويدجات التالية:
   - ويدجت حالة المهام (مقياس دائري)
   - ويدجت المهام حسب المسؤول (جدول)
   - ويدجت مؤشرات الخطة الاستراتيجية (بطاقات)
   - ويدجت نسب إنجاز المبادرات (شريط تقدم)

#### ح. إنشاء لوحة معلومات الإحصاءات العامة:

1. إنشاء صفحة لوحة معلومات جديدة:
   ```bash
   php artisan make:filament-page GeneralStatisticsDashboard --panel=admin
   ```

2. تطوير الويدجات التالية:
   - ويدجت عرض نمو المركز (رسم بياني خطي)
   - ويدجت نسب النجاح والإكمال (مقياس دائري)
   - ويدجت مقارنة الأداء بين الفروع (رسم بياني شريطي)
   - ويدجت الملخص العام للمؤشرات (بطاقات)

#### ط. تطوير التقارير المختلفة (PDF & Excel):

1. إنشاء مجلد للتقارير:
   ```
   app/Filament/Admin/Reports
   ```

2. تطوير أنواع التقارير التالية:
   - تقارير المساجد والحلقات (شهرية، فصلية، سنوية)
   - تقارير المعلمين والطلاب (تقدم الطلاب، تقييم المعلمين)
   - تقارير الرواتب (كشوفات رواتب، حوافز)
   - تقارير مالية (ميزانية، تدفقات نقدية)
   - تقارير الطلبات والخدمات (إحصائية، تحليلية)
   - تقارير التسويق ومؤشرات الأداء

3. استخدام مكتبات لتصدير التقارير:
   - `spatie/laravel-browsershot` لإنشاء ملفات PDF
   - `spatie/simple-excel` لإنشاء ملفات Excel

## البدء بالتنفيذ:

لنبدأ بتنفيذ الخطة من خلال إنشاء أول لوحة معلومات وهي لوحة معلومات المساجد والحلقات:

1. أولاً، سننشئ صفحة لوحة معلومات المساجد والحلقات:

```bash
php artisan make:filament-page MosquesAndCirclesDashboard --panel=admin
```

2. ثم سننشئ الويدجات المطلوبة لهذه اللوحة:

```bash
php artisan make:filament-widget MosquesByRegionWidget --panel=admin
php artisan make:filament-widget CirclesByTypeWidget --panel=admin
php artisan make:filament-widget CirclesPerMosqueWidget --panel=admin
php artisan make:filament-widget OccupancyRatesWidget --panel=admin
```

3. سنكرر هذه العملية لكل لوحة معلومات مطلوبة ثم نقوم بربط جميع لوحات المعلومات في قائمة التنقل الرئيسية.
